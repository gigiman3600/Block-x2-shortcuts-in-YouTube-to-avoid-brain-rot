# YouTube x2 Shortcuts Blocker (Anti Brain-Rot)

A powerful Tampermonkey script designed to disable YouTube's "hold-to-speed" gestures. This tool is built for those who want to maintain focus and avoid cognitive overload.

## Features
- **Total Spacebar Lockdown**: Prevents 2x speed when holding Space.
- **Mouse Gesture Block**: Disables 2x speed when long-clicking the video.
- **Visual Cleanup**: Removes the "2x" overlay and bezel for a cleaner experience.
- **20+ Layers of Protection**: Uses prototype hijacking and event interception.

## How to Install
1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension.
2. Click [here](https://github.com/gigiman3600/Block-x2-shortcuts-in-YouTube-to-avoid-brain-rot/blob/main/block-x2.user.js) to install the script (Note: link works after uploading).
3. Refresh YouTube and stay focused.
